oriStone
首先，这是一个极其简单的语言。以后会不断优化的。
基于java语言

要求：
1、运算
2、字符串处理
3、支持变量
4、控制语句
5、动态数据类型
6、句末分隔符“；”。也可省略。代码块中，最后一条语句的句尾分号可以省略。
7、不支持return 。最后一句的计算结果，就是整个程序的运行结果
8、注释\\

9、空行也是为一条空语句。换行符不会被当做一个空白符处理。不能中途换行。所以大括号不能换行
10、语句体必须要有大括号

问题：
1、正则表达式字面量
2、if语句的dangling-else问题。是一个二义语法。java是else与最近的一个if对应




架构：
词法分析（分割单词token对象，单词对应的类型，保存单词的类型，所在位置的行号等信息）
	标识符、整型字面量、字符串字面量。Token.EOF，用于表示程序的结束；Token.EOL，用于表示换行符。是String
语法分析
抽象语法树(AST)
（编译器、解释器）

AST:
使用token保存字段数据
BNF 巴克斯范式：(与正则表达式的区别是 在于具体是以单词为单位检查匹配还是以字符为单位检查)。包含非终结符
	factor:	 		NUMBER | "(" expression ")"
	term:			factor { ( "*" | "/" ) factor }
	expression:	term { ( "+" | "-" ) term }
